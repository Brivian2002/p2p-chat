How to Use

Your ID – Tap to copy. It stays the same on this device.
Add Contact – Tap + → Add Contact → enter their ID (optional nickname).
Start Chat – Tap a contact from the list.
Send Files – Click paperclip, choose any file (images/videos/audio/documents).
Voice Note – Click microphone to record, click again to stop and send.
React – Click the reaction button on any message to add emoji.
Delete Message – Long‑press (or right‑click) a message to delete it.
Edit Contact – Three‑dot menu on a contact → Edit.
Settings – Cog icon to change theme, online status, language.
